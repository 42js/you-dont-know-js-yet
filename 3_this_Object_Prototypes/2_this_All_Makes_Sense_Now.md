- `this` 란, 모든 함수의 스코프 내에서 자동으로 정의되는 특수 식별자 키워드다. (o)

- inner regular function 의 default binding: `window`/`undefined`

- 전역에서 `apply` , `call` , `binding` 을 이용한 function call
